# 人工智能训练师三级实践题整理

本项目用于整理**人工智能训练师三级**考试实践题的代码解析与学习笔记，帮助学生理解每道题的代码含义，避免死记硬背。

## 项目结构

```
.
├── README.md                 # 本文件
├── analysis/                 # 整理后的学习文档
│   ├── README.md             # 整理规范
│   ├── 1.1.1/README.md       # 第 1.1.1 题解析
│   ├── 1.1.2/README.md       # 第 1.1.2 题解析
│   └── ...                   # 其他题目解析
└── original-file/            # 原始考试资料
    └── 人工智能训练师三级上网素材/  # 40 道实践题原始文件
```

## 整理内容

每道题的解析文档包含：

1. **题目大意**：一句话概括这道题在做什么。
2. **空缺部分答案 + 代码解释**：逐空给出答案，并用通俗语言解释代码含义。
3. **用到的 Python 函数（仅空缺部分）**：列出考试中需要填写的函数/方法。
4. **需要背的单词/术语（仅空缺部分）**：列出需要记忆的英文单词和列名（如有）。
5. **注意事项**：易错点、关键理解点、考试容易丢分的地方。

## 使用说明

1. 打开 `analysis/` 下对应题号的目录。
2. 阅读 `README.md` 中的代码解释。
3. 对照原始资料 `original-file/人工智能训练师三级上网素材/<题号>/` 中的代码框架进行练习。
